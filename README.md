# NeonTech - E-Commerce Website

NeonTech is a modern and responsive E-Commerce Website built using HTML, CSS, and JavaScript.

## Features:
* Responsive Design
* Product Showcase
* Add to Cart Functionality
* Remove Items from Cart
* Shopping Cart Counter
* Checkout Flow
* Payment Method Selection
* Popup Notifications

## Technologies Used:
* HTML
* CSS
* JavaScript
